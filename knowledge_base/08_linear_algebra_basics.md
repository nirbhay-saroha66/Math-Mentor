# Linear Algebra Basics

## Matrix Operations
(A + B)ij = Aij + Bij
(AB)ij = Σk Aik Bkj
(AB)^T = B^T A^T

## Determinant
2×2: |a b; c d| = ad - bc
3×3: Use Sarrus rule or cofactor expansion

## Inverse
A^(-1) exists iff det(A) ≠ 0
For 2×2: A^(-1) = (1/det(A)) × [d -b; -c a]

## Eigenvalues
Solve det(A - λI) = 0
Eigenvector: (A - λI)v = 0

## Common Constraints
- Matrix multiplication: columns of A = rows of B
- Inverse: only for square matrices
