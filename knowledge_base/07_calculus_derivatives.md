# Derivatives - Rules and Formulas

## Basic Rules
d/dx(x^n) = nx^(n-1)
d/dx(e^x) = e^x
d/dx(ln x) = 1/x
d/dx(sin x) = cos x
d/dx(cos x) = -sin x
d/dx(tan x) = sec²x

## Chain Rule
d/dx(f(g(x))) = f'(g(x)) × g'(x)

## Product Rule
d/dx(uv) = u'v + uv'

## Quotient Rule
d/dx(u/v) = (u'v - uv')/v²

## Optimization
At local max/min: f'(x) = 0
Second derivative test: f''(x) > 0 → minimum, f''(x) < 0 → maximum
