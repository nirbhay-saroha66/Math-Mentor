# Inequalities and Absolute Value

## Absolute Value
|x| = x if x ≥ 0, -x if x < 0
|x| < a ⟺ -a < x < a (when a > 0)
|x| > a ⟺ x < -a or x > a (when a > 0)

## AM-GM Inequality
For non-negative a, b: (a + b)/2 ≥ √(ab), equality when a = b

## Common Inequality Rules
- Adding same value to both sides preserves inequality
- Multiplying by negative number reverses inequality
- Squaring both sides: only valid when both sides are non-negative

## Domain Constraints
When solving √(f(x)) > g(x): need f(x) ≥ 0 and consider g(x) sign
