# Matrices and Determinants - Extended

## Properties of Determinant
det(AB) = det(A) × det(B)
det(A^T) = det(A)
Swapping rows/columns negates determinant
Adding multiple of row to another doesn't change determinant

## Cramer's Rule
For Ax = b, x_i = det(A_i)/det(A) where A_i has column i replaced by b

## Rank
Rank = number of linearly independent rows/columns
For invertible n×n matrix: rank = n
