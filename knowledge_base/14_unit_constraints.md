# Unit and Domain Constraints

## Physical Quantities
- Length, area, volume: non-negative
- Probability: 0 to 1
- Angles: often 0 to 2π or -π to π

## Mathematical Domains
- √x: x ≥ 0
- ln(x), log(x): x > 0
- 1/x: x ≠ 0
- sin⁻¹(x), cos⁻¹(x): x ∈ [-1, 1]

## JEE Context
Always state domain when giving final answer. Reject solutions outside domain.
