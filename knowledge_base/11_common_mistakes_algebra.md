# Common Mistakes - Algebra

## Quadratic Equations
- Writing √(a²) = a (should be |a|)
- Forgetting ± in quadratic formula
- Not checking if solutions satisfy original equation

## Inequalities
- Multiplying by negative without flipping inequality
- Incorrectly squaring both sides of inequality

## Domain Errors
- √(x²) = x (wrong when x < 0)
- 1/(1/x) = x (correct but watch x=0)
- log(negative) is undefined
