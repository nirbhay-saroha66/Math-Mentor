# Probability Fundamentals

## Basic Definitions
P(A) = Number of favorable outcomes / Total outcomes (for equally likely)
0 ≤ P(A) ≤ 1
P(A') = 1 - P(A)

## Addition Rule
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
For mutually exclusive: P(A ∪ B) = P(A) + P(B)

## Conditional Probability
P(A|B) = P(A ∩ B) / P(B), when P(B) > 0

## Multiplication Rule
P(A ∩ B) = P(A) × P(B|A) = P(B) × P(A|B)
For independent events: P(A ∩ B) = P(A) × P(B)

## Bayes' Theorem
P(A|B) = P(B|A) × P(A) / P(B)
