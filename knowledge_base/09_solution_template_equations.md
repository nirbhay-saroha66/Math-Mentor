# Solution Template: Solving Equations

## Step-by-Step Approach
1. Identify equation type (linear, quadratic, polynomial, transcendental)
2. Rearrange to standard form
3. Apply appropriate method (factorization, formula, substitution)
4. Check all solutions against original equation (extraneous roots)
5. State final answer with domain constraints

## Verification
Always substitute back to verify. Watch for:
- Squaring both sides introducing extraneous roots
- Division by zero
- Domain restrictions (log, sqrt)
