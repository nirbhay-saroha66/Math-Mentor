# Sequences and Series

## Arithmetic Progression
nth term: a_n = a + (n-1)d
Sum: S_n = n/2 × (2a + (n-1)d) = n/2 × (first + last)

## Geometric Progression
nth term: a_n = ar^(n-1)
Sum (finite): S_n = a(1-r^n)/(1-r), r≠1
Sum (infinite): S = a/(1-r), |r| < 1

## Common Mistake
Using infinite GP sum when |r| ≥ 1 (diverges)
