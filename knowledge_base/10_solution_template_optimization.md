# Solution Template: Optimization Problems

## Steps for Max/Min
1. Identify quantity to optimize (objective function)
2. Express in single variable using constraints
3. Find domain of the variable
4. Compute derivative f'(x)
5. Find critical points: f'(x) = 0
6. Check endpoints of domain
7. Second derivative test or evaluate at all candidates
8. State answer with units

## Common Mistake
Forgetting to check boundary/endpoints when domain is closed interval
